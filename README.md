**Vibe Cadet Academy** is a high-stakes, gamified terminal simulation designed to demystify the world of **Agentic AI**, **CLI workflows**, and the **Model Context Protocol (MCP)**. 

Originally a simple CLI tutor, this "Elite Edition" has been re-architected into a narrative-driven training environment where students must restore a compromised space station by orchestrating autonomous AI agents.

## 🛠️ What's New in the Elite Edition?

We've evolved the curriculum from basic file manipulation to advanced AI systems architecture:

### 1. Agentic Orchestration Layer
* **Agent Monitoring:** Use `vibe status` to track the "Thinking Process" and "Context Window" of autonomous agents (Engineer & Captain).
* **Tool Authorization:** Practice the "Human-in-the-loop" security model by explicitly granting permissions (e.g., `vibe auth --agent Engineer --tool npm`).
* **Model Context Protocol (MCP):** Initialize standardized connections between LLMs and local data with `vibe mcp init`.

### 2. The "LLM-in-the-Middle" Workflow
* **Piping to Reasoning:** Learn to pipe raw system logs directly into an LLM for diagnostic analysis (`cat logs.txt | vibe explain`).
* **Structured Intelligence:** Experience why **JSON** is the language of agents by forcing structured outputs with the `--json` flag.

### 3. Advanced Terminal Mechanics
* **The `man` System:** A built-in field manual to teach self-sufficiency.
* **Live Telemetry (`tail -f`):** Real-time log streaming to simulate a living system.
* **Process Management:** Identify and terminate rogue AI processes using `ps` and `kill`.

### 4. Affective Learning & Gamification
* **Narrative Stakes:** A 13-phase mission to save Station Delta.
* **Visual Feedback:** ASCII art "Ghost in the Machine" detections and screen-shake "Glitch" effects on failure.
* **Proficiency Manifest:** A downloadable JSON certification upon completion, mapping specific CLI and AI competencies.

## 🎓 Learning Objectives (The L&D Perspective)

This tool is designed for AI educators and L&D professionals to move students past "Prompt Engineering" and into "AI Engineering":

1.  **CLI Fluency:** Master navigation, piping, and permissions.
2.  **AI Governance:** Understand the necessity of restricted tool access for agents.
3.  **System Orchestration:** Learn to manage multiple agents working toward a shared goal.
4.  **Debugging AI:** Identify hallucinations and use prompt governance to correct them.

## 💻 Tech Stack
* **Frontend:** Vanilla HTML5, CSS3 (with custom animations), and Javascript.
* **Design Pattern:** State-driven virtual file system (VFS).
* **Zero Dependencies:** Runs entirely in the browser—perfect for low-latency classroom environments.
